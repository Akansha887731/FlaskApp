To create database : db.create_all()

To add elementss in the table :-
    with app.app_context():
        user1 = User(username='ABC', email='ABC@gmail.com', password='dbjhsdjk')
        db.session.add(user1)
        db.session.commit()

To check all the elements in the table : User.query.all()

To check the first row only : User.query.first()

To filter out rows based on some condition : 
    User.query.filter_by(username='Akansha').all()
    User.query.filter_by(username='Akansha').first()

To fetch user with the specific id : User.query.get(1)

For adding posts for a user id :-
    with app.app_context():
        user = User.query.get(1)

        post1 = Post(title='Blog 1', content='This is the content of my first post.', user_id=user.id)
        post2 = Post(title='Blog 2', content='This is the content of my second post.', user_id=user.id)
        db.session.add(post1) 
        db.session.add(post2)
        db.session.commit()

        if user:
            for post in user.posts:
                print(post)
        else:
            print("No user with id 1 found.") 

To drop the database and create a new fresh :-

    with app.app_context():
            db.drop_all()
            db.create_all()

            print(User.query.all())
            print(Post.query.all())
